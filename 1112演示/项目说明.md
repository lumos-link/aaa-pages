# 拍卖平台高保真原型

## 项目概述

这是一个使用 HTML + Tailwind CSS + FontAwesome 构建的拍卖平台高保真原型，采用现代化的移动端优先设计，完全响应式布局。

## 技术栈

- **HTML5**: 语义化标签，结构清晰
- **Tailwind CSS**: 实用优先的CSS框架，快速构建现代化UI
- **FontAwesome 6.4**: 丰富的图标库
- **原生JavaScript**: 轻量级交互实现

## 项目结构

```
├── index.html              # 主入口文件，使用iframe嵌套其他页面
├── pages/                  # 页面目录
│   ├── home.html          # 首页 - 拍卖大厅
│   ├── detail.html        # 商品详情页
│   ├── category.html      # 分类浏览页
│   ├── publish.html       # 发布拍卖页
│   ├── my-auctions.html   # 我的拍卖页
│   ├── my-bids.html       # 我的出价页
│   ├── profile.html       # 个人中心页
│   └── auth.html          # 登录注册页
└── 项目说明.md            # 本文件
```

## 核心功能

### 1. 首页（home.html）
- 轮播横幅展示
- 分类筛选标签
- 热门拍卖品卡片列表
- 排序功能
- 响应式网格布局

### 2. 商品详情页（detail.html）
- 商品图片展示（支持缩略图切换）
- 商品基本信息
- 实时倒计时
- 出价功能
- 出价历史记录

### 3. 分类浏览页（category.html）
- 分类图标展示
- 筛选功能（价格、状态、排序）
- 商品列表展示

### 4. 发布拍卖页（publish.html）
- 完整的发布表单
- 图片上传功能
- 表单验证

### 5. 我的拍卖页（my-auctions.html）
- 拍卖列表展示
- 状态筛选（全部/进行中/已结束/草稿）
- 拍卖数据统计

### 6. 我的出价页（my-bids.html）
- 出价记录列表
- 状态筛选（全部/领先中/已出局/已成交）
- 快速再次出价

### 7. 个人中心页（profile.html）
- 用户信息展示
- 数据统计
- 账户设置入口

### 8. 登录注册页（auth.html）
- 登录/注册切换
- 表单验证
- 密码显示/隐藏

## 设计特点

### 色彩系统
- **主色**: #1a365d (深蓝色) - 专业、可信赖
- **辅助色**: #e53e3e (红色) - 价格、重要提示
- **背景色**: #f7fafc (浅灰) - 舒适阅读

### 响应式设计
- **移动端**: < 768px - 单列布局，底部导航
- **平板**: 768px - 1024px - 双列布局
- **桌面**: > 1024px - 多列布局，顶部导航

### UI特点
- 卡片式设计，阴影效果
- 悬停动画效果
- 清晰的视觉层次
- 现代化的圆角和间距

## 使用方法

1. **直接打开**: 在浏览器中打开 `index.html` 即可查看
2. **使用Live Server**: 推荐使用VS Code的Live Server插件，支持热更新
3. **本地服务器**: 使用任何HTTP服务器（如Python的http.server）
